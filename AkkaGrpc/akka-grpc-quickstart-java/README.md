# Akka gRPC QuickStart

- Generated from https://developer.lightbend.com/start/?group=akka&project=akka-grpc-quickstart-java

## Maven

- Ensure maven is installed

For windows

```shell
choco install maven
```

- Run below command before opening in IntelliJ

```shell

mvn install
```
